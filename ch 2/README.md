# Chapter 2: Conditional Statements

This folder contains Python programs demonstrating conditional statements (if, elif, else):

- **big.py**: Compares two numbers to find the larger one.
- **elif.py**: Uses multiple elif conditions for age-based eligibility (note: has some logical issues).
- **gender.py**: Greets users based on gender input.
- **if.py**: Checks if a number is even or odd.
- **leap.py**: Determines if a year is a leap year.
- **temp.py**: Categorizes temperature into ranges.
- **vote.py**: Checks voting eligibility based on age.

These programs illustrate basic decision-making in Python.
